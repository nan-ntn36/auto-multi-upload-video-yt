2. Config file 
CHROME_PATH: nơi chứa chrome đã cài đặt 
CLIENT_SECRET_FILE: là cái file khi setup https://console.cloud.google.com/
PROFILES: các profile nằm trong C:\Users\<Tên user của mình>\AppData\Local\Google\Chrome\User Data
	+ Mặt định là : defautl 
	+ Các profile khác : profile 1,2,3....
PROFILE_PORTS: mỗi một trang profile là 1 port khác nhau -> nó phải giống với trong file CLIENT_SECRET_FILE ( sửa ở cuối)
